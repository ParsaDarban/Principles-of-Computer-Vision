The original image is smoothed using a bilateral filter to retain edges while softening colors. Finally, the edges are combined with the smoothed image using a bitwise operation, resulting in a stylized, cartoon-like output.

The block size (9) specifies the size of the local neighborhood used for thresholding, and the constant 9 is subtracted from the mean to fine-tune the thresholding.

The bilateral filter reduces noise while preserving sharp edges, making it ideal for stylized processing like cartoonification. The parameter 9 specifies the diameter of the pixel neighborhood used for filtering. The sigmaColor controls how strongly colors in the neighborhood influence each other, with higher values resulting in smoother color transitions. The sigmaSpace determines the spatial influence, where larger values smooth areas over greater distances.
